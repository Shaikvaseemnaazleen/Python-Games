# Task Manager

Task Manager is a Web Application to manage our daily tasks.

## Functionalities

* Add Tasks
* Edit Tasks
* Delete Tasks

## Technical Stack

* Flask Web Framework
* HTML
* CSS
* SQLAlchemy
